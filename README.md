**Note:** this project is currently under aggressive development, please _do not_ use it in a production application.

This project provides API gateway for microservices. It features:

Features:
- Configurable parameters
- Auto-navigation support
- Built for modern frameworks
- Abstracted data model

Planned Features:
- Restrict input to specific character sets
- Configurable UI elements
- Grouping support
